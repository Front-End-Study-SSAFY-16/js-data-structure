### 제출 방식
1. 챕터 폴더 아래 본인 이름의 폴더를 생성하고 소스 파일 업로드(소문자 권장)<br>
ex) chap_01/jongwoo/quick-sort.js<br>
2. 각 챕터별로 실행 결과 화면 캡처본을 README.md를 생성하여 추가
3. 커밋 컨벤션은 feat:, fix:, docs: 등의 prefix와 영문으로 작성하는 것을 적극 권장
